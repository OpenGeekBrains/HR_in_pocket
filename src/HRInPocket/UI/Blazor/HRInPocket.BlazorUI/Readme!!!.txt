Для запуска проекта требуется провести некоторые телодвижения:

1) В первую очередь - надо либо обновить базу данных IdentityServer (IdentityServerConfigDB), либо просто удалить её (воссоздастся автоматически)
2) Запускать нужно 2 проекта одновременно. При чем первым должен стартовать IdentityServer, а клиент (HRInPocket.BlazorUI) - последним.
   В середину можно всталять запуск каких-нибудь других API если они будут
3) Чтобы работала аутентификация через внешние поставщики аутентификационных данных (Google, VK, FaceBook) - необходим файл секретов для
   приложения HRInPocket.IdentityServer (не входит в репозиторий), скачать который можно в нашей группе в telegram

   ПКМ на проекте HRInPocket.IdentityServer => Управление секретами пользователей => откроется обычный json, в который нужно скопировать содержимое файла HRInPocket.IdentityServer.json
